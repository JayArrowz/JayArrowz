About Jay Arrowz

I'm a software engineer who loves building practical, well-crafted libraries and tools that solve real problems for developers. My work spans across the .NET ecosystem—from SignalR documentation generators to high-performance bit packing libraries—as well as the React and TypeScript world with authentication component libraries.

I maintain several open-source packages on NuGet and npm, and I'm always looking to collaborate on projects that push developer tooling forward. When I'm not coding, I'm probably exploring a new problem domain or tinkering with a weekend project that might just turn into the next useful tool.

I'm open to collaboration on open-source C# projects, NuGet packages, and developer tooling. Feel free to reach out via GitHub or support my work on Buy Me a Coffee.
